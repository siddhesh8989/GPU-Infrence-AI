Build a full-stack, production-level project titled:

"GPU-Optimized Neural Network Inference Pipeline"

The system must simulate and implement a real-world AI inference pipeline with advanced optimization techniques inspired by research-level systems.

---

## 🔷 CORE REQUIREMENTS

1. BACKEND (FastAPI preferred)

* Use Python FastAPI server
* Implement REST APIs:

  * /upload (image/video input)
  * /predict (run inference)
  * /metrics (return performance stats)
* Support async processing using asyncio

---

2. MACHINE LEARNING MODEL

---

* Use at least TWO models:

  1. Image Classification (ResNet / MobileNet using PyTorch)

  * Random Forest classifier (scikit-learn) for comparison or hybrid inference

* Allow switching between:

  * CPU mode
  * Optimized (simulated GPU mode)

3. PREPROCESSING MODULE

---

* Resize images
* Normalize pixel values
* Convert to tensors
* Handle batch inputs

4. DYNAMIC BATCHING SYSTEM (IMPORTANT)

5. MEMORY & ASYNC OPTIMIZATION

---

* Simulate:

  * Data loading + inference
* Avoid blocking operations

6. GPU INFERENCE SIMULATION (VERY IMPORTANT)

---

* If GPU not available:
